## Rev Orgs

### Available Operations

* [Create](/rev_orgs/create) - Creates a Rev organization in the authenticated user's Dev
organization.

* [Delete](/rev_orgs/delete) - Deletes the Rev organization.
* [Get](/rev_orgs/get) - Retrieves the Rev organization's information.
* [Post Get](/rev_orgs/post_get) - Retrieves the Rev organization's information.
* [List](/rev_orgs/list) - Gets the list of Rev organizations' information belonging to the
authenticated user's Dev Organization which the user is also authorized
to access.

* [Post List](/rev_orgs/post_list) - Gets the list of Rev organizations' information belonging to the
authenticated user's Dev Organization which the user is also authorized
to access.

* [Update](/rev_orgs/update) - Updates the Rev organization's information.
