## Works

### Available Operations

* [Create](/works/create) - Creates new work ([issue](https://devrev.ai/docs/product/build),
[ticket](https://devrev.ai/docs/product/support)) item.
[task](https://docs.devrev.ai/product/tasks) and opportunity work types
are supported in the beta version.

* [Delete](/works/delete) - Deletes a work item.
* [Export](/works/export) - Exports a collection of work items.
* [Post Export](/works/post_export) - Exports a collection of work items.
* [Get](/works/get) - Gets a work item's information.
* [Post Get](/works/post_get) - Gets a work item's information.
* [List](/works/list) - Lists a collection of work items.
* [Post List](/works/post_list) - Lists a collection of work items.
* [Update](/works/update) - Updates a work item's information.
